# Prediagnóstico Aura Desarrollos 2026

## Qué es este proyecto
Resumen ejecutivo del prediagnóstico operativo realizado por London Consulting Group para Aura Desarrollos (Altamira, Tamaulipas) en Abril 2026. Es una página web publicada en GitHub Pages que resume las áreas de oportunidad, el plan de transformación y el retorno sobre la inversión.

## Quién es el usuario
Gerardo Tamayo — consultor en London Consulting Group. Trabaja en el proyecto de transformación operativa de Aura Desarrollos. No es desarrollador — usa Claude Code para construir y actualizar la página.

## Estructura de la página
1. Header + Stats (CMM 1.5, 50 oportunidades, 19 entrevistas, 13 pilares)
2. 5 Áreas de Intervención (introducción ejecutiva)
3. Cards de problemas por área:
   - Planeación y Presupuesto (CMM 1.2) — tabla comparativa
   - Compras y Abastecimiento (CMM 1.0) — tabla comparativa
   - Ejecución y Control de Obra (CMM 1.1) — tabla comparativa + Curva S visual
   - Ingeniería y Diseño (CMM 2.5) — tabla comparativa
   - Estructura, PMO y Gobierno (CMM 1.0)
   - Comercial y Desarrollo de Negocios (CMM 3.0)
   - Entrega y Calidad (CMM 1.0)
   - Soporte: Finanzas, Capital Humano y TI
4. Lo Que Hay Que Hacer (4 ejes de transformación)
5. Playbook (3 urgencias, 5 fases, 10 entregables, inicio Mayo 2026)
6. Sala de Control Digital
7. Inversión y ROI (3 escenarios: conservador 5%, medio 7.5%, optimista 10%)

## Cómo publicar cambios
Después de editar `index.html`:
```
git add index.html
git commit -m "descripción del cambio"
git push
```
Luego recargar la página con Ctrl+Shift+R (tarda ~30 segundos en actualizar).
